# Dogecoin (DOGE) — No Formal Whitepaper

Dogecoin was created in December 2013 by Jackson Palmer and Billy Markus as a joke/meme cryptocurrency.
It is a fork of **Luckycoin**, which is a fork of **Litecoin**, which is a fork of **Bitcoin**.

No formal whitepaper was ever published. The closest references are:

1. **Bitcoin Whitepaper** (technical basis): https://bitcoin.org/bitcoin.pdf
2. **Dogecoin Manifesto**: https://foundation.dogecoin.com/manifesto/
3. **Dogecoin Foundation Trailmap**: https://foundation.dogecoin.com/trailmap/prologue/
4. **Source code**: https://github.com/dogecoin/dogecoin
